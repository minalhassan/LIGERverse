# 🦁 LIGERverse: Professional CTF & Pentesting Platform

![LIGERverse Banner](https://github.com/minalhassan.png)

**LIGERverse** is a state-of-the-art, interactive cybersecurity learning platform designed for CTF players, bug bounty hunters, and penetration testers. It transforms traditional cheatsheets into a dynamic, hands-on experience.

---

## 🚀 Key Features

- **🛡️ 7 Core Hacking Modules:** 
  - System Hacking (Nmap, SMB, RPC)
  - Web Hacking (Recon, Fuzzing, SQLi, XSS)
  - File Hacking (Data extraction, Forensics)
  - Cryptography (Encoding, Ciphers)
  - Forensics (PCAP, Steganography)
  - Password Cracking (Hashcat, John)
  - Privilege Escalation (Linux/Windows)
- **🛠️ Integrated Tool Explorer:** A centralized database of 150+ commands with smart search.
- **📚 Practice Labs:** 50+ curated free TryHackMe rooms categorized by domain.
- **💻 Terminal Simulations:** Realistic command-line simulations for learning tool outputs.
- **🔍 Global Intel Search:** Press `/` to instantly find any tool or technique.
- **⚡ High-End UX:** Built with Next.js 14, Tailwind 4, and Framer Motion for a premium hacker aesthetic.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 🏁 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/minalhassan/LIGERverse.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
